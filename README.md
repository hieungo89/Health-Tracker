## How to Get Started ##
- Install dependencies in Client:
  1. cd into client
  2. `npm install`
- Install dependencies in Server:
  1. cd into server
  2. `npm install`
- Run Servers:
  1. In client - `npm run start`
     - client run on PORT 8080
  2. In server - `npm run server`
     - server runs on PORT 3000
- Launch site: http://localhost:8080
